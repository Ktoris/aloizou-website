# Antonis Loizou & Associates — Website Rebuild

Modern rebuild of [aloizou.com.cy](http://www.aloizou.com.cy/) using **TanStack Start**, **TanStack Router**, **React 19**, and **Tailwind CSS v4**.

## Pages

- `/` — Home
- `/about` — The firm
- `/services` — Practice areas
- `/cyprus` — Cyprus market
- `/contact` — Enquiry form
- `/sitemap.xml` — Sitemap (SSR)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Production

```bash
npm run build
npm run preview
```

## Stack

- Navy + gold editorial design (Cormorant Garamond + Inter)
- File-based routing in `src/routes/`
- SSR via TanStack Start
